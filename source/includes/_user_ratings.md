# User Ratings

## User Rating Create

```python
import requests

requests.get('https://api.madefire.com/api/reader/user/rating/')
```

```shell
curl "https://api.madefire.com/api/reader/user/rating/"
```

```javascript
```


Allowed Methods: POST, OPTIONS
